# Kidney-Disease-Detection


# How to Run the Code

### Steps to Run the Code

Clone the repository to your local machine:

```bash
https://github.com/tanay4343/Kidney-Disease-Detection
``` 

### Steps 01-Create a conda environment after opening the terminal

```bash
conda create -n my_venv python=3.12 -y
```

### Steps 02-Activate the conda environment

```bash
conda activate my_venv
```

### Steps 03-Install the required libraries

```bash 
pip install -r requirements.txt
```
